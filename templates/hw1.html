Homework 1

영화 오픈/좌석 취소 알림 서비스

학번: 202001052 | 이름: 김채린

1) Problem Statement Canvas

고객

  • 인기 영화·특별 상영회·IMAX/4DX 등을 수요가 많은 영화를 예매하려는 관람객
  • 좋은 자리를 확보하고 싶은 연인, 가족, 친구 단위 관람객
  • 매진으로 인해 좌석을 확보하지 못해 불편을 겪는 영화 팬덤

컨텍스트

  • 인기 영화/특별관은 예매 오픈 직후 빠르게 매진됨
  • 예매 오픈 시간을 알기 어려워 오픈 여부를 자주 확인해야 함
  • 영화 시작 20분 전까지 무료 취소가 가능 → 언제든 취소표가 나올 수 있으나 알 수 없음
  • 상영 종류마다 선호 좌석이 다르지만, 인기 좌석 정보가 없음

근본 문제

  • 언제 예매가 열리고 원하는 좌석이 언제 비는지 알 수 없음
  • 좌석 경쟁 치열도를 예측할 수 없어 전략적 선택이 불가능

감정적 영향

  • 원하는 자리 확보 실패 → 실망, 좌절, 불공정성 느낌
  • 소중한 관람 기회(데이트/팬덤 이벤트 등)를 놓친다는 불안감
  • 원하는 좌석을 얻지 못했을 때의 아쉬움과 스트레스

정량적 영향

  • 예매/취소 여부 확인을 위해 수시로 사이트 접속 필요
  • 만족스럽지 않은 자리를 원했던 자리와 동일 비용 지불
  • 원하는 시간대 예매 실패 → 다른 시간대/날짜로 관람 고려

대안 & 단점

  • 공식 앱/웹 수동 새로고침 → 알림 없음, 시간 소모 큼
  • 커뮤니티 정보 → 정확성/속도 부족, 이미 매진일 가능성
  • 비공식 대리 예매 → 불법, 추가 비용/보안 위험

2) 솔루션 (3줄)

  • 영화 예매 오픈 알림: 원하는 영화가 오픈되면 알림
  • 좌석 취소 알림: 원하는 좌석이 비면 알림
  • 빅데이터 기반 예측: 예매 패턴 분석으로 인기 좌석 사전 안내

3) 기능 명세

  • 원하는 영화, 상영관 입력
  • 원하는 매진된 좌석 입력
  • 영화 오픈 시 메일 알림
  • 좌석 취소 시 메일 알림
  • 사람이 몰릴 것 같은 좌석 예측

4) 간단한 UX 흐름

  • 회원가입/로그인
  • 영화 오픈 알림 신청 (알림 인원 수, 인기 좌석 안내)
  • 좌석 취소 알림 신청 (알림 인원 수 표시)
  • 내 페이지 (알림 신청 내역)
